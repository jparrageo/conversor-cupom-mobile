CONVERSOR DE CUPOM MOBILE - ANA STORE

ANDROID
1. Extraia o ZIP.
2. Abra conversor_cupom_mobile.html pelo Chrome.
3. Mantenha a internet ligada.
4. Selecione o PDF da venda.
5. Toque em "1. Ler PDF".
6. Confira e corrija a prévia.
7. Toque em "2. Gerar cupom em PDF".
8. Compartilhe o PDF gerado pelo WhatsApp.

IPHONE
O iPhone pode bloquear páginas HTML locais. Tente:
1. Salvar o HTML no app Arquivos.
2. Compartilhar > Abrir no Safari.
3. Se o Safari apenas exibir o código ou não executar, será necessário usar uma hospedagem estática.

IMPORTANTE
- O PDF é processado no aparelho.
- A internet serve apenas para carregar PDF.js e jsPDF.
- O conversor foi ajustado para o modelo atual da Ana Store.
- Sempre confira a prévia.
